# Shapeshifter DSL — v1 Specification (Frozen)

**Status:** Frozen 2026-04-18. This document captures the language as it
existed when the Agent Harness was bootstrapped. Do not edit this file.
See `dsl/spec.md` for the current version.

---

## Host

Python 3. Expressions are Python `list` or `tuple` objects. The interpreter
is `src/interpreter.py` (`ShapeshifterInterpreter`).

## Atoms

- **Numbers / booleans**: returned as-is (`42`, `True`, `False`).
- **Strings**: looked up in the active environment; if not found, returned as
  the string literal itself (enabling string constants).
- **Lists/tuples**: evaluated as function calls or special forms.

## Standard Primitives (in `_default_env`)

| Symbol | Type | Notes |
|---|---|---|
| `add` `sub` `mul` `div` | arithmetic | Python `operator.*` |
| `gt` `lt` `eq` | comparison | return Python `bool` |
| `not` | boolean | `operator.not_` |
| `print` | I/O | host `print` |
| `list` `first` `rest` `cons` | list ops | standard |

## Special Forms

### `['quote', expr]`
Returns `expr` unevaluated. Enables treating code as data.

### `['if', cond, true_branch, false_branch]`
Standard conditional.

### `['set', name, val_expr]`
Binds `name` in the current environment. At top level, binds globally.

### `['defn', name, [params], body]`
Named function definition. Equivalent to `set name (lambda params body)`.

### `['lambda', [params], body]`
Anonymous closure with lexical scoping.

### `['begin', expr1, ..., exprN]`
Evaluates all expressions in order; returns the last value.

### `['and', expr1, ..., exprN]`
Short-circuiting conjunction. Returns `False` on first falsy result;
otherwise returns the last value.

### `['or', expr1, ..., exprN]`
Short-circuiting disjunction. Returns the first truthy result;
otherwise returns `False`.

### `['dict-get', dict_expr, key_expr]`
Extracts a value from a Python dict by key. Raises `TypeError` if
`dict_expr` does not evaluate to a dict.

## Safety: The Cage

### `['run_with_gas', limit, expr]`
Evaluates `expr` with a local step budget of `limit`. Raises
`RecursionError` on exhaustion. Automatically switches to a
`StrictEnv` (capability-gated) environment containing only whitelisted
primitives. The global hard ceiling is 500 steps (`max_steps`).

## Observability

Primitives `get_metrics` (injected by experiments) returns a dict of OTel
counters readable from within the DSL via `dict-get`.

## Known Limitations at Freeze

- `local_max` is not propagated through named function dispatch; lambdas
  stored in `global_env` run against the global ceiling, not a local cage.
  Fix tracked as DEF-004 (resolved as of v1).
- Host callables in `global_env` outside the whitelist are reachable by
  mutated expressions unless `run_with_gas` is used.
